⚙️ Configura tu URL de Google Apps Script para sincronizar

¿Cómo conectar con Google Sheets?

1
Abre Google Sheets y crea una hoja nueva

Dale un nombre como "Mis Gastos 2025".

2
Ve a Extensiones → Apps Script

Se abrirá el editor de código de Google.

3
Pega el código del archivo apps-script.gs

Está en el repositorio junto a este archivo.

4
Implementar → Nueva implementación

Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier usuario. Copia la URL.

5
Pega esa URL arriba y listo

Toca "Sincronizar" y los gastos van directo a tu hoja.
