Configuración de Google Sheets

⚙️ Configura tu URL de Google Apps Script para sincronizar

¿Cómo conectar con Google Sheets?

1
Abre Google Sheets y crea una hoja nueva

Dale un nombre como "Mis Gastos 2025".

2
Ve a Extensiones → Apps Script

Se abrirá el editor de código de Google.

3
Borra el código existente y pega el de apps-script.gs

El archivo está junto a este HTML. Cópialo completo.

4
Haz clic en Implementar → Nueva implementación

Tipo: "Aplicación web". Ejecutar como: "Yo". Acceso: "Cualquier usuario". Copia la URL que aparece.

5
Pega esa URL arriba y listo

Cada vez que sincronices, los gastos van directo a tu hoja.
